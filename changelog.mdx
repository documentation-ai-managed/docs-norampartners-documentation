## Recent Updates

<Update label="2024-10-20" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time shipment tracking API endpoint for consumer electronics fulfillment.
- Introduced automated kitting configurations for media products.

## Improvements

- Enhanced returns grading accuracy by 20% with AI-powered inspection.
- Optimized B2C fulfillment workflows for faster order processing.

```javascript
// New tracking endpoint example
const trackShipment = async (shipmentId) => {
  const response = await fetch(`https://api.norampartners.com/v2/track/${shipmentId}`, {
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
  });
  return response.json();
};
```

</Update>

<Update label="2024-09-15" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched asset recovery dashboard for partners in consumer goods markets.
- Integrated 3PL returns repair service with grading reports.

## Breaking Changes

- Updated authentication to require `{API_KEY}` in headers instead of query params.
- Deprecated legacy `/v1/fulfill` endpoint; migrate to `/v2/fulfill`.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Updated auth header
  headers: { "Authorization": `Bearer ${YOUR_API_KEY}` }
  ```
  ```python
  # Updated auth header
  headers = {"Authorization": f"Bearer {YOUR_API_KEY}"}
  ```
</CodeGroup>

</Update>
